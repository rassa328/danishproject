En stor del av det danska ordförrådet kan du *räkna ut* från svenskan med några enkla motsvarigheter. Men det är **stavning, inte uttal** – och ibland landar regeln på ett ord som betyder något annat.

## Vanliga motsvarigheter

| Svenska | Danska | Exempel |
| --- | --- | --- |
| ä | æ | äpple → <span lang="da">æble</span> |
| ö | ø | öl → <span lang="da">øl</span>, kött → <span lang="da">kød</span> |
| å | å | (samma) |
| -t | -d | mat → <span lang="da">mad</span>, gata → <span lang="da">gade</span>, ut → <span lang="da">ud</span> |
| ck | k | mycket → <span lang="da">meget</span>, socker → <span lang="da">sukker</span> |

## Varning – regeln gäller inte alltid

- Många ord är oregelbundna: mjölk → <span lang="da">mælk</span> (inte *mølk*), flicka → <span lang="da">pige</span>, höst → <span lang="da">efterår</span>.
- **Dold falsk vän:** det danska <span lang="da">høst</span> finns – men betyder *skörd* i vardagsspråk, inte höst.
- Och kom ihåg: <span lang="da">mad</span> och <span lang="da">gade</span> *stavas* med d men uttalas med blødt d (se lektionen om talad danska).

> Tumregel: använd motsvarigheterna för att *gissa*, men slå upp ordet om betydelsen är viktig – en regel kan leda dig rakt in i en falsk vän.
