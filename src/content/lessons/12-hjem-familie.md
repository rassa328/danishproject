Som svensk läser du danska bostadsannonser nästan obehindrat – men just därför sätter de små glidningarna åt. Det farligaste ordet i hela temat är <span lang="da">stue</span>.

## Den stora fällan: stue

<span lang="da">stue</span> betyder **vardagsrum** – och i sammansättningar **våning** (våningsplan). Det betyder *inte* svensk "stuga".

| Danska | Betyder | Inte att förväxla med |
| --- | --- | --- |
| <span lang="da">stue</span> | vardagsrum | stuga = <span lang="da">sommerhus / hytte</span> |
| <span lang="da">stueetage</span> | bottenvåning | — |
| <span lang="da">første sal</span> | en trappa upp (våning 1) | obs: *inte* bottenvåningen |
| <span lang="da">værelse</span> | rum (sov-/barnrum) | jfr <span lang="da">rum</span> = (även) utrymme |

> <span lang="da">Vi bor i stueetagen, og børnene har hvert sit værelse.</span> = Vi bor på bottenvåningen, och barnen har varsitt rum. Tänk aldrig "stuga" när du ser <span lang="da">stue</span>.

## Bolig och rum

En **bostad** heter <span lang="da">bolig</span>, en **lägenhet** <span lang="da">lejlighed</span>. Rummen:

| Danska | Svenska |
| --- | --- |
| <span lang="da">køkken</span> | kök |
| <span lang="da">badeværelse</span> | badrum |
| <span lang="da">soveværelse</span> | sovrum |
| <span lang="da">have</span> | trädgård |
| <span lang="da">nabo</span> | granne |

Lägg märke till att rummen byggs med <span lang="da">-værelse</span>, inte "-rum": ett **sovrum** är ett <span lang="da">soveværelse</span>.

## Familjen

| Danska | Svenska |
| --- | --- |
| <span lang="da">forældre</span> | föräldrar |
| <span lang="da">søskende</span> | syskon |
| <span lang="da">svigermor</span> | svärmor |
| <span lang="da">barnebarn</span> | barnbarn |
| <span lang="da">fætter</span> | (manlig) kusin |
| <span lang="da">kusine</span> | (kvinnlig) kusin |

Danskan skiljer på kön i ordet "kusin": en manlig kusin är <span lang="da">en fætter</span>, en kvinnlig <span lang="da">en kusine</span>. Och <span lang="da">svigermor</span> bygger på <span lang="da">sviger-</span> precis som svenskans "svär-": <span lang="da">svigerfar</span>, <span lang="da">svigerinde</span> (svägerska).
