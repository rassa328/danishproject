Bra nyheter: nästan all dansk grammatik fungerar som svensk. Lägg energin på de få punkter som faktiskt skiljer sig.

## 1. *da* eller *når* för "när"?

Svenskans *när* motsvaras av två danska ord:

- **<span lang="da">da</span>** – om **dåtid**, en specifik gång: <span lang="da">Da jeg var barn …</span> (= "den gången då").
- **<span lang="da">når</span>** – om **nutid/framtid** eller upprepning: <span lang="da">Når jeg kommer hjem …</span>

## 2. "Börja" finns inte

Svenskans *börja* heter <span lang="da">begynde</span> (dåtid <span lang="da">begyndte</span>) eller mer vardagligt <span lang="da">starte</span>:

- <span lang="da">Jeg begyndte at lære dansk sidste år.</span> = Jag började lära mig danska förra året.

(Notera också: danskan har inget reflexivt "mig" – <span lang="da">at lære</span>, inte "lära *mig*".)

## 3. Perfekt med *er* vid rörelse

Vid rörelseverb använder danskan ofta <span lang="da">er</span> (vara) i stället för <span lang="da">har</span>:

- <span lang="da">Jeg er gået.</span> = Jag har gått (i väg).
- <span lang="da">Han er rejst.</span> = Han har rest.

Men för de flesta andra verb: <span lang="da">har</span>, precis som svenskans *har*.

## 4. Snabba småskillnader

- *att* + infinitiv = <span lang="da">at</span> (<span lang="da">at lære</span>), men uttalas som "å" – lätt att blanda ihop med <span lang="da">og</span> ("och"), som också låter som "å".
- *fortfarande* = <span lang="da">stadig</span> (en falsk vän – inte "stadig/stabil").

> Lägg inte tid på att leta efter stora grammatiska skillnader – de finns knappt. Lyssna och läs mycket, så fastnar mönstren av sig själva.
