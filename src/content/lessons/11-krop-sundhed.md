När du blir sjuk på en resa till Danmark behöver du ord som inte alltid ser ut som du tror. Det mesta känns igen, men notera artikeln (en/et) och ett par stavningsfällor.

## Kroppen

De flesta kroppsord är lätta att läsa – men artikeln (en/et) snubblar man lätt på, och stavningen skiljer sig ibland från svenskan.

| Danska | Svenska | Anmärkning |
| --- | --- | --- |
| <span lang="da">hoved</span> | huvud | et-ord; stumt <span lang="da">d</span>, uttalas ungefär "houe" |
| <span lang="da">hals</span> | hals | en-ord; <span lang="da">ondt i halsen</span> = ont i halsen (precis som svenskan) |
| <span lang="da">mave</span> | mage | en-ord; <span lang="da">g</span> blir <span lang="da">v</span>: <span lang="da">ondt i maven</span> |
| <span lang="da">ryg</span> | rygg | en-ord; bara ett <span lang="da">g</span> |
| <span lang="da">ben</span> | ben | et-ord; betyder *både* benet du går på och knoglen, exakt som svenska "ben" |
| <span lang="da">knæ</span> | knä | et-ord; svenskt <span lang="da">ä</span> blir <span lang="da">æ</span> |

> Stavningsfälla: svenskt *g* blir ofta danskt *v* eller försvinner – <span lang="da">mage → mave</span>, <span lang="da">rygg → ryg</span>. Och slut-*d* i <span lang="da">hoved</span> hörs inte alls.

## Symtom

| Danska | Svenska |
| --- | --- |
| <span lang="da">at have ondt</span> | att ha ont |
| <span lang="da">feber</span> | feber |
| <span lang="da">hoste</span> | hosta (både substantiv och verb) |
| <span lang="da">forkølet</span> | förkyld |
| <span lang="da">syg</span> | sjuk |

Lägg märke till att man säger <span lang="da">at have ondt i</span> + kroppsdel: <span lang="da">"Jeg har ondt i hovedet"</span> = jag har ont i huvudet.

## Hos lægen

En läkare heter <span lang="da">en læge</span> – inte *lægare*. Det danska <span lang="da">at lege</span> betyder däremot *att leka*, så blanda inte ihop dem.

| Danska | Svenska | Anmärkning |
| --- | --- | --- |
| <span lang="da">læge</span> | läkare | en-ord; ≠ <span lang="da">lege</span> (leka) |
| <span lang="da">medicin</span> | medicin | en-ord |
| <span lang="da">recept</span> | recept | **et-ord**: <span lang="da">et recept</span> – inte *en recept* |
| <span lang="da">sygdom</span> | sjukdom | en-ord |
| <span lang="da">sundhed</span> | hälsa | en-ord; ordagrant "sundhet", men svenskan säger *hälsa* |
| <span lang="da">blodtryk</span> | blodtryck | et-ord; bara ett <span lang="da">k</span> |

En vardaglig replik hos läkaren: <span lang="da">"Jeg er forkølet og har ondt i halsen."</span> – Jag är förkyld och har ont i halsen.
