En kväll på dansgolvet ser bekant ut för en svensk – men just danssnacket är fullt av falska vänner. En <span lang="da">dansetime</span> är ingen timme utan en lektion, <span lang="da">sjov</span> betyder inte lugn, och <span lang="da">hurtig</span> handlar om fart, inte humör. <span lang="da">Salsa</span>, <span lang="da">tango</span> och <span lang="da">en dans</span> är identiska på båda språken – dem kan du redan. Här tar vi det från första <span lang="da">trin</span> till sista svängom.

## Ta en danskurs

Redan anmälan gömmer en genusfälla, och lektionsordet lurar örat. Håll koll på <span lang="da">kursus</span> och <span lang="da">time</span>.

| Danska | Svenska | Anmärkning |
| --- | --- | --- |
| <span lang="da">et kursus</span> (pl. <span lang="da">kurser</span>) | kurs | et-ord! Svenskan säger *en* kurs |
| <span lang="da">en dansetime</span> | danslektion | en-ord. FALSK VÄN: <span lang="da">time</span> = lektion (och timme) |
| <span lang="da">en begynder</span> / <span lang="da">øvet</span> | nybörjare / van, erfaren | <span lang="da">begynder</span> en-ord; <span lang="da">øvet</span> ~ sv. *övad*: <span lang="da">øvede dansere</span> = vana dansare |
| <span lang="da">et trin</span> | (dans)steg | et-ord. <span lang="da">trin</span> = steg! <span lang="da">et dansetrin</span> = ett danssteg |
| <span lang="da">at øve sig</span> | att öva | -a→-e, ö→ø – men reflexiven förblir <span lang="da">sig</span>, inte norskans *seg* |

<span lang="da">"Hvornår begynder timen?"</span> – När börjar lektionen?

> **Stavningsfälla:** svenska verb på *-a* blir *-e* på danska, och vokalerna byter skepnad: dansa → <span lang="da">danse</span>, träna → <span lang="da">træne</span>, öva → <span lang="da">øve</span> (ä→æ, ö→ø).

## På dansgolvet och ut på stan

Ute på <span lang="da">dansegulvet</span> (golv → <span lang="da">gulv</span>, o→u) gäller det att kunna bjuda upp – och att inte gå på kvällens värsta fälla.

| Danska | Svenska | Anmärkning |
| --- | --- | --- |
| <span lang="da">at byde op til dans</span> | att bjuda upp | bjuda → <span lang="da">byde</span>. Bildligt: <span lang="da">byde nogen op til dans</span> = utmana någon |
| <span lang="da">Må jeg få denne dans?</span> | Får jag lov? | da. <span lang="da">må</span> = får (tillåtelse)! |
| <span lang="da">en natklub</span> | nattklubb | en-ord. Dubbelkonsonanten ryker två gånger: <span lang="da">nat</span> + <span lang="da">klub</span> |
| <span lang="da">et par dansesko</span> | ett par dansskor | plural oböjt: <span lang="da">sko</span>, inte "skor" (<span lang="da">en dansesko</span>, en-ord) |
| <span lang="da">en kjole</span> | klänning | en-ord. FALSK VÄN: sv. kjol heter <span lang="da">en nederdel</span> |
| <span lang="da">sjov, sjovt</span> | rolig, kul | FALSK VÄN! da. <span lang="da">rolig</span> = lugn – se nedan |

<span lang="da">"Skal vi gå i byen bagefter?"</span> – Ska vi gå ut (på stan) efteråt? Idiomet <span lang="da">gå i byen</span> betyder ordagrant "gå i staden".

> **Falska vänner:** <span lang="da">sjov</span> betyder **rolig, kul**, medan danskans <span lang="da">rolig</span> betyder **lugn** – rakt tvärtom mot vad örat tror (samma rolig-fälla som i *falske venner*). <span lang="da">"en rolig aften"</span> = en lugn kväll, men <span lang="da">"en sjov aften"</span> = en rolig, kul kväll.

## Stilar och rörelse

Stilnamnen (<span lang="da">folkedans</span>, <span lang="da">ballet</span>) gissar du lätt – men tempo-ordet <span lang="da">hurtig</span> lurar, och magen dyker upp igen.

| Danska | Svenska | Anmärkning |
| --- | --- | --- |
| <span lang="da">hurtig</span> | snabb | FALSK VÄN: sv. hurtig = pigg, käck |
| <span lang="da">en mavedans</span> | magdans | en-ord. <span lang="da">mave</span> (mage → <span lang="da">mave</span>, g→v) |
| <span lang="da">en opvarmning</span> | uppvärmning | en-ord. upp → <span lang="da">op</span> |

Och håll <span lang="da">takten</span> – <span lang="da">i takt</span> eller <span lang="da">ude af takt</span>.

> **Uttal:** danskt blødt d mjukar upp slutljudet nästan till engelskt "th": <span lang="da">fod</span> [ˈfoˀð] (fot) sväljs nästan helt. Och slut-g:et tystnar – <span lang="da">hurtig</span> låter ungefär som "hurti".

## Prata om dans

När dansen är slut ska den berömmas. Här sitter en sista genusfälla: <span lang="da">publikum</span> är ett et-ord.

| Danska | Svenska | Anmärkning |
| --- | --- | --- |
| <span lang="da">flot</span> | snygg, elegant | opakt ord värt att lära: <span lang="da">"du danser flot!"</span> |
| <span lang="da">en konkurrence</span> | tävling | en-ord. da. <span lang="da">konkurrence</span> = tävling (även konkurrens); <span lang="da">en dansekonkurrence</span> = en danstävling |
| <span lang="da">et publikum</span> | publik | ET-ORD! Svenskan säger *en* publik |

<span lang="da">"Hvor danser de flot!"</span> – vad snyggt de dansar! – ropar <span lang="da">publikum</span> på en <span lang="da">dansekonkurrence</span>.

> **Stavningsfälla:** dubbeltecknad konsonant förenklas i danskan – samma regel som gjorde <span lang="da">nat</span> + <span lang="da">klub</span> av *nattklubb* tidigare: rygg → <span lang="da">ryg</span>, och <span lang="da">flot</span> mot svenskans *flott*.

Kulturkoll: <span lang="da">Vild med dans</span> är danskarnas *Let's Dance* (på TV 2) – och <span lang="da">at være vild med noget</span> betyder att vara tokig i något. På gymnasiets <span lang="da">gallafest</span> dansar sistaårseleverna <span lang="da">Les Lanciers</span>, en gammal turdans som i dag nästan bara lever kvar i Danmark.
