På danska arbetsplatser möter du många ord som ser svenska ut men som beter sig annorlunda. Här är de viktigaste – och de tre värsta fällorna.

## Grundord på jobbet

| Danska | Svenska |
| --- | --- |
| <span lang="da">arbejdsplads</span> | arbetsplats |
| <span lang="da">kollega</span> | kollega |
| <span lang="da">chef</span> | chef |
| <span lang="da">ansat</span> | anställd |
| <span lang="da">løn</span> | lön |
| <span lang="da">opgave</span> | uppgift |
| <span lang="da">møde</span> | möte |
| <span lang="da">ansøgning</span> | ansökan |

Notera stavningen: svenskt *lön* blir <span lang="da">løn</span>, och *möte* blir <span lang="da">møde</span> med blødt d (uttalas nästan som ett mjukt "l"/"th"-ljud, inte hårt d).

## Tre falska vänner du MÅSTE kunna

> Tre arbetsord lurar svenskar varje dag: <span lang="da">ferie</span> betyder *semester/ledighet*, <span lang="da">frist</span> betyder *deadline*, och <span lang="da">fyre</span> betyder *avskeda*. Förväxla dem inte!

- <span lang="da">ferie</span> är inte svenskans "ferie" (lov från skolan). På danska är det **semester** generellt: <span lang="da">Jeg holder ferie i juli.</span> = Jag har semester i juli.
- <span lang="da">frist</span> är inte svenskans "frist" (anstånd/uppskov). Det betyder **deadline, tidsgräns**: <span lang="da">Fristen er på fredag.</span>
- <span lang="da">at fyre</span> låter som "fyra" eller "fira" men betyder **avskeda**: <span lang="da">Han blev fyret.</span> = Han fick sparken.

## Arbetstid och frånvaro

| Danska | Svenska |
| --- | --- |
| <span lang="da">fuldtid</span> | heltid |
| <span lang="da">deltid</span> | deltid |
| <span lang="da">overarbejde</span> | övertid |
| <span lang="da">vikar</span> | vikarie |
| <span lang="da">at sygemelde sig</span> | sjukanmäla sig |

Är du sjuk ringer du och <span lang="da">sygemelder</span> dig. Behöver en kollega ersättas tar en <span lang="da">vikar</span> över. Och jobbar du för mycket blir det <span lang="da">overarbejde</span> – danskan säger inte "overtid" om jobb.
